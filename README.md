# Path of Crafting

A collection of tools for Path of Exile crafting research.


## Tools

### Recorder

Watches your clipboard during alteration spam sessions and records every magic item tooltip to a `.pocr` file for later statistical analysis.

```
poc.recorder.exe <output_file>
```

Use standard `Ctrl+C` copy in PoE. Start the recorder, spam alts, close it when done.

More tools coming.

## Building

```
xmake build
```

Requires LLVM and C++23. Windows only.
